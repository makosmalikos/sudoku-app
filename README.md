# sudoku-app
Sudoku App is a web application that makes solving Sudoku more fun and interactive. It is designed for students, puzzle lovers, and anyone who wants to improve their logical thinking skills.

The most unique feature of the app is Scan Sudoku, which allows users to take a photo of a Sudoku puzzle from a book or notebook and convert it into a digital board. The app also includes different difficulty levels, hints, dark mode, a timer, and a Daily Challenge leaderboard.

I created this project to make Sudoku easier to access and more engaging for modern users. It helps people practice problem-solving and concentration in a simple and enjoyable way.

How to Play

1. Choose a difficulty level.
2. Click **New Game** to load the selected Sudoku puzzle.
3. Select a cell and enter numbers using the number buttons.
4. Use hints, undo, and other tools if needed.
5. Complete the puzzle as quickly and accurately as possible to achieve the best score.

Unique Features

1. 5 difficulty levels (Easy to Master)
2. Dark Mode
3. Hint system
4. Daily Challenge Leaderboard
5. Undo system
6. Delete 
7. Scan Sudoku – users can take a photo of a Sudoku puzzle from a book, newspaper, or notebook. The application automatically scans the puzzle and transfers it into a digital Sudoku grid, making it easier to solve and track progress.
8. Unlock the Cat(photo): players reveal pieces of a hidden image after completing levels
